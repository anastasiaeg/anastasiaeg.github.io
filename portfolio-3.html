Rare Disease Patient Portal

Description

  • github.com/anastasiaeg/RareDiseasePatientPortal

  • August 2017 - December 2017

A portal for patients with rare diseases to be able to learn more about the disease, estimate the lost opportunity & treatment costs using Python, Flask, nltk, SQLite.

Developed for Data-Driven Decision Making Class.
